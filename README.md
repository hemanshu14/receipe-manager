# Project Description
This application provides you a set of REST APIs to create, update, fetch, delete & fetch (based on a filter request) receipes from a H2 database.

# How to start the project?
Clone the code from (https://github.com/hemanshu14/receipe-manager) and import it as a maven project in the IDE like Intellij or Eclipse.
- Run the Application as a Spring Boot App from the IDE or run the command from the terminal: mvn spring-boot:run
- You can view the API documentation here - http://localhost:8080/swagger-ui/index.html

# More info about the project
- There is a POSTMAN collection also given which has all the REST API details with examples.
